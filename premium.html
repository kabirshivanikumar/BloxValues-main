Common questions

How do I get my code after paying?

After completing checkout through the payment link, you'll receive your unlock code from the payment provider's receipt/email or the confirmation page, depending on how the site owner set up checkout. If you don't receive one, contact the site owner directly.

Does this sync across devices?

Unlocking currently applies to the browser you redeem the code in. Use the same code again in another browser to unlock it there too — each code can be redeemed, but ask the site owner if you need multi-device support.

Can I get a refund?

Refunds are handled by the site owner through whichever payment method was used at checkout — reach out to them directly.
